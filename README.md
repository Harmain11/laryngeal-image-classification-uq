# Laryngeal Image Classification with Uncertainty Quantification

## Overview
This notebook demonstrates the classification of laryngeal images using Support Vector Machines (SVM). It includes data loading, preprocessing, model training, hyperparameter tuning with grid search, and evaluation. The notebook also performs uncertainty quantification via bootstrap resampling and visualizes class probability distributions and prediction results.

## Features
- Loading and preprocessing of laryngeal image dataset
- SVM classification with flattened image features
- Hyperparameter tuning using GridSearchCV
- Performance evaluation with accuracy, precision, recall, F1-score, and confusion matrix
- Uncertainty quantification using bootstrap for confidence intervals
- Visualization of classification probabilities and prediction outcomes

## Tech Stack
- Python
- Jupyter Notebook / Google Colab
- Libraries: numpy, pandas, matplotlib, seaborn, scikit-learn, OpenCV (cv2)

## How to Use
1. Upload your laryngeal image dataset to Google Drive.
2. Modify the notebook paths to point to your dataset locations.
3. Run the notebook cells sequentially in Google Colab or Jupyter Notebook.
4. Review the classification reports, uncertainty quantification results, and visualizations.
